# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for Station. Each ADR documents a significant technical decision: what was decided, why, and what the consequences are.

ADRs are written at the time the decision is made, when the context is fresh. They serve as a record for future contributors (including future-you) who need to understand why the architecture is the way it is.

## Format

Station uses the [MADR](https://adr.github.io/madr/) (Markdown Architectural Decision Records) format.
